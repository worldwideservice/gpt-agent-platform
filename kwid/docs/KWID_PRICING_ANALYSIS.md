# Анализ тарифов KWID

## ✅ Наблюдения с KWID

### Динамическое изменение цены при изменении количества ответов:

**1,000 ответов:**
- Launch: $18/месяц ✅ (доступен)
- Scale: $45/месяц
- Max: $90/месяц

**5,000 ответов:**
- Launch: $18/месяц ❌ (недоступно для этого количества)
- Scale: $205/месяц
- Max: $350/месяц

**15,000 ответов:**
- Launch: $18/месяц ❌ (недоступно)
- Scale: $578/месяц (текущий план)
- Max: $973/месяц

**20,000 ответов:**
- Launch: $18/месяц ❌ (недоступно)
- Scale: $760/месяц
- Max: $1,280/месяц

### Формула расчета цены (приблизительно):

**Launch:**
- Фиксированная цена: $18 (не зависит от количества ответов)
- Доступен только для 1,000 ответов

**Scale:**
- Базовая цена: ~$30-40
- Множитель: ~$0.03-0.04 за ответ
- Формула: `basePrice + (responses * multiplier)`

**Max:**
- Базовая цена: ~$50-60
- Множитель: ~$0.04-0.05 за ответ
- Формула: `basePrice + (responses * multiplier)`

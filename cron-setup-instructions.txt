# Инструкция по настройке cron для бэкапов

Выполните вручную:

1. Откройте crontab:
   crontab -e

2. Добавьте эти строки в конец файла:

# Backup Supabase database daily at 2:00 AM
0 2 * * * cd "/Users/maksimgolovaty/Library/Mobile Documents/com~apple~CloudDocs/Development/AI agent" && source scripts/.backup-secrets.sh && ./scripts/backup-database-cron.sh >> logs/backup.log 2>&1

# Cleanup old backups (older than 30 days) at 2:30 AM
30 2 * * * cd "/Users/maksimgolovaty/Library/Mobile Documents/com~apple~CloudDocs/Development/AI agent" && ./scripts/cleanup-old-backups.sh 30 >> logs/backup.log 2>&1

3. Сохраните и закройте редактор (в nano: Ctrl+X, затем Y, затем Enter)

4. Проверьте:
   crontab -l | grep backup

Готово! Бэкапы будут создаваться ежедневно в 2:00 AM.

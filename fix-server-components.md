# Исправление ошибок Server Components

## Проблема
Next.js пытается prerender (статически генерировать) страницы и API routes, которые должны быть динамическими.

## Решение
1. API routes должны быть динамическими
2. Страницы с 'use client' не должны prerender
3. Страницы с async серверными компонентами должны быть динамическими

description: Применить миграции базы данных
---

Примени миграции базы данных для проекта.

Проверь наличие Supabase настроек и выполни:
1. Проверь доступность Supabase (SUPABASE_URL из .env.local)
2. Примени миграции: `npx supabase db push` или используй скрипт из /scripts/migrations/

Покажи результат миграций и список применённых изменений.

Собери проект GPT Agent Platform для production.

Выполни следующие команды:
1. `npm run type-check` - проверка типов TypeScript
2. `npm run lint` - проверка кода
3. `npm run build` - сборка Next.js приложения

Покажи результат каждого шага. Если есть ошибки, предложи их исправить.

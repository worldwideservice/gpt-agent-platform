---
description: Исправить импорты и зависимости
---

Проверь и исправь импорты в проекте.

Выполни:
1. Найди неиспользуемые импорты
2. Найди отсутствующие зависимости
3. Проверь circular dependencies
4. Исправь путь импортов (относительные vs абсолютные)
5. Запусти `npm run lint -- --fix` для автоматического исправления

Покажи список исправленных файлов.

---
description: Проверить переменные окружения
---

Проверь конфигурацию переменных окружения для проекта.

Выполни:
1. Проверь наличие `.env.local` файла
2. Запусти `npm run verify:env` (если доступно)
3. Сравни с `env.example` и покажи недостающие переменные
4. Проверь критичные переменные:
   - NEXTAUTH_SECRET
   - SUPABASE_URL
   - SUPABASE_SERVICE_ROLE_KEY
   - OPENROUTER_API_KEY
   - REDIS_URL

Покажи статус каждой критичной переменной (установлена/отсутствует).

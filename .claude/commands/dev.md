description: Запустить development окружение
---

Запусти development окружение для GPT Agent Platform.

Проверь:
1. Существует ли `.env.local` файл
2. Установлены ли зависимости (node_modules)
3. Запусти `npm run dev` или `make dev` в фоновом режиме

Покажи URL приложения (обычно http://localhost:3000) и любые важные логи при запуске.

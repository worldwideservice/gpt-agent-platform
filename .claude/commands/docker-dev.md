---
description: Запустить Docker dev окружение
---

Запусти полное Docker development окружение для проекта.

Выполни:
1. Проверь наличие Docker и Docker Compose
2. Запусти `docker compose -f docker-compose.dev.yml up --build` в фоновом режиме
3. Покажи статус запущенных контейнеров
4. Покажи доступные URL для сервисов (Next.js, API, Redis, Supabase)

Если есть ошибки при запуске, предложи их исправить.

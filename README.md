# Стартовый каркас

Минимальный проект на Next.js (App Router) с авторизацией через NextAuth.

## Что внутри
- Next.js 14 + App Router
- Tailwind CSS для базовой стилизации
- Простейшая авторизация по аккаунту из переменных окружения
- Пустая страница рабочего пространства `/workspace`

## Быстрый запуск
```bash
npm install
npm run dev
```

### Учётные данные
По умолчанию используются значения:
- Email: `demo@example.com`
- Пароль: `Demo1234!`

Можно переопределить переменными окружения:
```
AUTH_DEFAULT_EMAIL=you@example.com
AUTH_DEFAULT_PASSWORD=SuperSecret123
AUTH_DEFAULT_NAME=Ваше имя
```

## Дальнейшие шаги
Структура очищена от старых модулей. Можно добавлять нужные сущности, API и описание архитектуры с нуля.
